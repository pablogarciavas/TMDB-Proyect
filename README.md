# MovieDB - The Movie Database App

Aplicación web para explorar películas usando la API de The Movie Database (TMDB).

## Características

- Búsqueda en tiempo real de películas, actores, directores y estudios
- Filtrado por géneros
- Detalles completos de películas (cast, trailers, proveedores de streaming)
- Watchlist personal
- Películas próximas a estrenar
- Juego de adivinanza de películas
- Diseño responsive

## Tecnologías

- React 18
- TypeScript
- Vite
- Tailwind CSS
- GSAP
- Axios
- TMDB API

## Scripts

- `npm run dev` - Servidor de desarrollo
- `npm run build` - Build de producción
- `npm run preview` - Preview de producción
